# 内购挑选助手

一个用来处理每月内购商品清单的浏览器小工具。导入 Excel 后，可以按品牌、类型、价位和预算筛选商品，加入购物车或候选清单，最后复制订购单、下载 CSV，或者导出分享图，方便自己挑选，也方便和同事一起核对。

## 主要功能

- 导入 `.xlsx`、`.xls`、`.csv` 商品清单
- 自动识别商品名、品牌、规格、价格、库存、图片等字段
- 自动合并重复商品，并在导入检查里提示合并数量
- 导入后检查异常：缺品牌、缺价格、缺图片、库存未知、分类为“其他”
- 异常检查支持一键定位问题商品
- 支持多关键词搜索，例如 `徽章 不鸭 -贴纸`
- 按品牌、类型、价位、预算筛选商品
- 常用品牌可以星标置顶
- 支持奶油柔和、浅色、深色三种配色
- 支持紧凑、舒适、大图三种卡片密度
- 三栏宽度可以拖动调整，也可以一键恢复默认布局
- 商品可以加入购物车、候选清单和对比小桌
- 购物车按品牌分组，并支持一键折叠/展开品牌组
- 预算浮窗支持胶囊模式、展开查看、拖动位置
- 支持复制订购单、下载 CSV、导出分享图
- 支持自定义分类关键词，方便每月新清单继续复用
- 支持手机端/窄屏浏览，底部有快捷入口

## 怎么使用

1. 打开网页。
2. 点击左侧「选择文件」。
3. 选择本月的商品清单 Excel 或 CSV 文件。
4. 查看导入检查弹窗，必要时点击「定位」查看问题商品。
5. 根据需要筛选品牌、类型、价位和预算。
6. 点击商品卡片空白处加入购物车，暂时不确定的可以先放进候选清单。
7. 选好后，在「输出清单」里复制订购单或下载 CSV。
8. 如果需要发给别人确认，可以展开「分享图和导出预览」，填写署名并导出分享图。

## 搜索技巧

搜索框支持多个关键词，用空格隔开。

```text
徽章 不鸭
```

表示同时包含“徽章”和“不鸭”。

```text
徽章 -贴纸
```

表示包含“徽章”，但排除包含“贴纸”的商品。

## 数据和隐私

这个工具是纯前端网页，商品清单会在浏览器本地读取和处理。

也就是说：

- 导入的 Excel 不会自动上传到服务器
- 购物车、候选清单、预算、主题、常用品牌、自定义分类等偏好会保存在当前浏览器本地
- 换电脑或换浏览器后，本地保存的数据不会自动同步
- 如果页面状态异常，可以在「其他操作」里点击「清理本机数据」恢复默认状态

## 发布和更新

这个项目可以直接用 GitHub Pages 托管。

更新网页时，只需要把 GitHub 仓库里的 `index.html` 替换成最新版，然后提交 `Commit changes`。GitHub Pages 会自动重新部署。

当前版本已经加入页面版本检查。新版发布后，网页会尽量自动绕开浏览器缓存加载最新版本。如果仍然看到旧页面，可以尝试刷新一次，或者在网址后加一个版本参数，例如：

```text
?v=55
```

## 常见问题

### 导入 Excel 后图片不显示

图片能否显示取决于清单里的图片链接是否允许外部访问。如果原图链接需要登录、过期了，或者禁止外部网页读取，就可能无法显示。

### 分类不准确怎么办

可以在左侧最底部的「其他操作」里展开「分类维护」，给未识别商品补充关键词。补充后的关键词会保存在当前浏览器本地，下次导入清单时继续生效。

### 页面状态乱了怎么办

可以在「其他操作」里点击「清理本机数据」。这会清空当前浏览器保存的购物车、候选清单、预算、主题、常用品牌和自定义分类。

## 适合谁用

适合每月面对一大张内购商品清单、需要控制预算、筛选品牌、整理订购单的人使用。

尤其适合：

- 自己慢慢挑
- 和同事一起核对
- 给朋友发一张确认图
- 每月复用类似格式的商品清单
